<leader> <G> = Gmail

BRIGHTNESS:
<leader> <up> = brightness up
<leader> <down> = brightness down

Volume control:


<leader> <Lower Volume> = Change output to headphones
<leader> <Raise Volume> = Change output to speakers

Display:

<Alt> <H/J/K/L> = Control window position 1/2
<Alt> <shift> <H/J/K/L> = Control window position on 1/4
<Alt> <1> = Window mode/fullscreen
<Alt> <2> = Minimize window
<leader> <1> = Swap to workspace 1
<leader> <2> = Swap to workspace 2 (and so on)

<ctrl> <Alt> <1> = Swap window to workspace 1
<ctrl> <Alt> <2> = Swap window to workspace 2
<ctrl> <Alt> <3> = Swap window to workspace 3 (and so on)


Screenshoots:

<ctrl> <Alt> <X> = Screenshoot windowed (No popup, just copy) 
<shift> <KP/> = Screenshoot windowed (Popup, no copy) 
<shift> <KP*> = Screenshoot the application (Popup, no copy)
